# Terminal Texas Hold'em

A lightweight, text-based Texas Hold'em poker engine built in Python. This game brings the classic casino experience to your command line, featuring a full game loop, a chip-based wallet system, and an automated hand evaluator.

## 🃏 Features

* **Full Game Loop:** Play through the Pre-Flop, Flop, Turn, and River.
* **Hand Evaluation:** Automatically calculates the winner based on poker hand rankings (High Card up to Four of a Kind).
* **Wallet System:** Track your chips and the CPU's chips across multiple rounds.
* **Interactive CLI:** Clear terminal prompts and formatted card visuals.
* **Paced Gameplay:** Built-in delays to mimic the feel of a real dealer.
